## 🚀 Start Here: 3 Quick Steps

### Step 1: Install & Run Locally (2 minutes)
```bash
npm install
npm run dev
```
Open http://localhost:3000

### Step 2: Update Your Information (10 minutes)
Edit these files with your information:
- [components/sections/Hero.tsx](components/sections/Hero.tsx) - Name, quote, links
- [components/sections/About.tsx](components/sections/About.tsx) - Summary
- [components/sections/Experience.tsx](components/sections/Experience.tsx) - Work history
- [components/sections/Skills.tsx](components/sections/Skills.tsx) - Your skills
- [components/sections/Tools.tsx](components/sections/Tools.tsx) - Technologies
- [components/sections/Certifications.tsx](components/sections/Certifications.tsx) - Certs
- [components/sections/WeekendProjects.tsx](components/sections/WeekendProjects.tsx) - Projects
- [components/Footer.tsx](components/Footer.tsx) - Social links
- [app/layout.tsx](app/layout.tsx) - Meta information

### Step 3: Deploy to Vercel (5 minutes)
```bash
git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/portfolio
git push -u origin main
```
Then import project to [vercel.com](https://vercel.com)
